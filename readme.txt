NullPay Plugin for WooCommerce.

== Description ==

NullPay WooCommerce Plugin. You can receive payment through your personal account with Automation.

== Features ==
* Digital Product
* Fee System
* Block Checkout
* And many more...

== Supported Gateways ==
* bKash
* Rocket
* Nagad
* Upay
* Cellfin
* Tap
* Ok Wallet
* Paypal
* Stripe
* Paddle
* Perfect Money
* Cryptomus
* Binance Personal (C2C)
* etc
